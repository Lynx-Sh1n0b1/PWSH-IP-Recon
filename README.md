# PWSH-IP-Recon
Powershell IP recon script
**************************
$ Launch PS script.
$ Insert IP address.

Scrip uses:
- ipinfo.io
- ping (alive\dead)
- nslookup
- Resolve-DnsName
- whois
- you also can use traceroute function (just uncomment traceroute part. I've commented it bkz traceroute take some time and I hate wait)
